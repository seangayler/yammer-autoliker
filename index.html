Yammer Autoliker

How this works

This tool automatically likes posts on a Yammer page. It works by saving a program to your bookmarks bar that runs when you click on that bookmark. For browser security reasons, you need to add the bookmark manually. It is assumed that your internet connection is sufficiently fast.

Different programs

There are two programs available:

  • Program 1: One that likes every post in the page's history.
  • Program 2: One that likes every currently loaded post on the page.

User guide

You may need to first display your bookmarks bar in your browser if it is currently hidden. You need to copy the text representing each program into the URL section of the bookmark.

  1. Add a new bookmark in your browser of choice
  2. Copy the program text below from the "Get the programs" section
  3. Paste the program text into the URL section of the new bookmark
  4. Click the bookmark while on a Yammer page to execute the program
